## 🎯 使用建议

### 学习路径
```
1. ⭐ 阅读 CODING_STANDARDS.md（必读）
   ↓
2. 阅读 HOW_TO_ADD_LEETCODE.md（第一次）
   ↓
3. 实践添加一道题目
   ↓
4. 熟悉后使用 QUICK_REFERENCE.md（日常使用）
```

- **添加新题目**: 查看 QUICK_REFERENCE.md 的 5 步流程

- **查找代码片段**: 在 QUICK_REFERENCE.md 中搜索
- **解决配置问题**: 查看 CONFIG_README.md 的常见问题部分
